- DEC Tracker sovereign wallet (NC/WC dual currency)
- HORTON mesh node with hardware fingerprint registration

- FFF Shopping Circles for combined buying power

## System Requirements
- **Windows**: Windows 10 or later
- **Mac**: macOS 11 (Big Sur) or later
- **Linux**: Ubuntu 20.04+ or equivalent
- **Browser**: Chrome 90+ (for extension)
- **Node.js**: 18+ (for running from source)

## Known Limitations
- Wallet balances are demonstration values until bank account wiring is complete
- NIC flashing requires administrator/root privileges
- Chrome extension requires Developer Mode enabled
